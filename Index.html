Trilha de Sistemas de Software

O que é a Trilha de Sistemas de Software?

A trilha de Sistemas de Software concentra-se no desenvolvimento, projeto e manutenção de sistemas de software complexos, escaláveis e de alta qualidade.

Esta especialização aborda tanto aspectos teóricos quanto práticos da construção de software, com ênfase em:

  • Engenharia de software e metodologias ágeis
  • Arquitetura de software e padrões de projeto
  • Desenvolvimento web e mobile
  • Banco de dados e sistemas de informação
  • Testes automatizados e qualidade de software
  • DevOps e integração contínua

Os estudantes desta trilha aprendem a construir sistemas robustos que atendem às necessidades dos usuários e se adaptam às mudanças tecnológicas.

Por que escolhi essa trilha?

Minha escolha pela trilha de Sistemas de Software foi motivada por vários fatores:

  • Paixão por desenvolvimento: Desde meu primeiro contato com programação, sempre me fascinei pelo processo de criar soluções através de código.
  • Impacto tangível: Sistemas de software têm o poder de transformar a maneira como pessoas e organizações funcionam no dia a dia.
  • Oportunidades profissionais: O mercado para desenvolvedores de software continua aquecido e em constante evolução.
  • Desafios técnicos: Construir sistemas escaláveis, seguros e eficientes representa desafios intelectuais estimulantes.

Durante meus estudos, participei de projetos práticos que me mostraram a importância de uma boa arquitetura de software e de práticas de desenvolvimento modernas. Um projeto em particular, onde desenvolvemos um sistema de gerenciamento acadêmico, me fez perceber como o software bem projetado pode simplificar processos complexos.

Além disso, acredito que a especialização em Sistemas de Software me dará as ferramentas necessárias para contribuir com projetos de código aberto e, eventualmente, criar minhas próprias soluções inovadoras.

Algumas disciplinas desta trilha que mais me interessam:

  • Engenharia de Software (MAC0466)
  • Banco de Dados (MAC0439)
  • Programação para Web (MAC0346)
  • Programação de Software Básico (MAC0344)
  • Tópicos Avançados em Programação Orientada a Objetos (MAC0447)
  • Desenvolvimento de Aplicações Móveis (MAC0470)
